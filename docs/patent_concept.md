# Patent Concept: Autonomous Swarm Drone Park Maintenance System

## Invention Title
Multi-Modal Swarm Drone Coordination Architecture for Selective Environmental Maintenance

## Core Innovation
A hub-led autonomous drone swarm system that:
- Uses AI vision for selective task identification
- Applies PSO for collision-free multi-agent path planning
- Coordinates heterogeneous drone roles (scout, cleaner, trimmer, waterer, patrol)
- Operates across seasons with adaptive task logic

## Key Claims
1. A system for autonomous park maintenance using coordinated drone swarms
2. A PSO-based task assignment algorithm for heterogeneous drone role delegation
3. A hub-based recharging and waste collection architecture
4. An AI vision layer for real-time environmental object detection from drone-mounted cameras

## Technical Differentiators
- Role-heterogeneous swarm (not just identical drones)
- Centralised hub with pipe-based waste transport
- Night-time patrol mode with light-show formation
- Seasonal adaptive logic (rain, summer, winter modes)
