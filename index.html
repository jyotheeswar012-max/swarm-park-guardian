How It Works

One complete autonomous cycle — from detection to clean park.

1

Scout & Detect

Scout drones patrol all 5 park zones using YOLOv8 computer vision. They identify debris, grass growth, water waste, and fallen branches with 94%+ accuracy.

2

PSO Task Assignment

The central hub receives detections and runs Particle Swarm Optimization to assign the nearest available drone of the right role — zero conflicts, no collisions.

3

Execute & Collect

Cleaner, Trimmer, and Waterer drones navigate to targets via computed waypoints. Waste is transported back to the hub through a pipe-docking system.

4

Recharge & Repeat

Low-battery drones autonomously return to hub for solar recharging. The cycle runs 24/7, across all seasons — rain, summer, and winter modes.

5

Night Patrol + Light Show

After dark, patrol drones switch to LED formation mode — securing the perimeter while creating a visual light display above the park.

Patent Concept

An original invention filed under the inventor's name.

Multi-Modal Swarm Drone Coordination Architecture for Selective Environmental Maintenance

This system represents a novel invention combining heterogeneous drone swarms, AI object detection, PSO-based task delegation, and a centralised hub with pipe-based waste collection — operating autonomously across all seasons with zero human intervention.

Inventor: A. Jyotheeswar Reddy  |  Jurisdiction: India (IPO)  |  Category: Robotics + AI Systems

  • 01 Autonomous swarm coordination using heterogeneous drone roles (scout, cleaner, trimmer, waterer, patrol)
  • 02 PSO-based real-time task assignment algorithm for collision-free multi-agent navigation
  • 03 Hub-based centralised waste collection via pipe-docking architecture
  • 04 AI vision layer (YOLOv8) for environmental object detection from drone-mounted cameras
  • 05 All-season adaptive operation with night patrol and LED light-show formation mode
